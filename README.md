# 🏦 Bharat Growth & Investment Bank (BGI)

A Secure Digital Banking System built using Python and Streamlit.

## Features

- Create Account
- Login System
- Deposit Money
- Withdraw Money
- View Account Details
- Forgot PIN

## Tech Stack

- Python
- Streamlit
- JSON Database

## Run Project

Install dependencies

pip install streamlit

Run the project

streamlit run app.py
